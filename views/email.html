Send Email Endpoint

POST /send-email

Request Body Parameters:

to Recipient email address (required)
subject Email subject line (required)
text Plain text content of the email (required)
html HTML content of the email (optional)

Example Request:

{ "to": "recipient@example.com", "subject": "Test Email", "text": "This is a test email", "html": "

This is a test email

With HTML formatting

" }

Example Response:

{ "success": true, "message": "Email sent successfully" }

Response Codes

200 Success Email sent successfully
400 Bad Request Missing required parameters
500 Server Error Failed to send email

Important Notes

  • All fields except 'html' are required
  • The 'html' field is optional but recommended for better email formatting
  • Emails are sent using NodeMailer with SMTP
  • Make sure to handle the response appropriately in your application
