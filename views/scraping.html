Web Scraping API Documentation

Web Scraping

POST /scraping

Request Body:

url URL website yang akan di-scrape (required)

Example Request:

{
  "url": "https://example.com"
}

Example Response:

{
  "title": "Example Website",
  "description": "Website description from meta tags",
  "links": [
    {
      "url": "https://example.com/page1",
      "text": "Link Text"
    }
  ],
  "images": [
    {
      "url": "https://example.com/image.jpg",
      "alt": "Image description"
    }
  ],
  "text": [
    "Content from paragraph 1",
    "Content from paragraph 2"
  ]
}

Notes:

  • URL harus valid dan dapat diakses
  • Response akan berisi data yang berhasil di-scrape dari website target
  • Semua gambar dan link akan diambil beserta atributnya
  • Teks akan diambil dari elemen paragraph (p)

Response Codes:

200 Success - Data berhasil di-scrape dari website
400 Bad Request - URL tidak valid atau tidak tersedia
500 Server Error - Gagal melakukan scraping website
