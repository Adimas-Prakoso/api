Endpoint Details

Web Scraping

POST /scraping

Request Body Parameters:

url URL of the website to scrape (required)

Example Request:

{ "url": "https://example.com" }

Example Response:

{ "title": "Example Website", "description": "Website description from meta tags", "links": [ { "url": "https://example.com/page1", "text": "Link Text" } ], "images": [ { "url": "https://example.com/image.jpg", "alt": "Image description" } ], "text": [ "Content from paragraph 1", "Content from paragraph 2" ] }

Response Codes

200 Success Data successfully scraped from website
400 Bad Request Invalid URL or URL not provided
500 Server Error Failed to scrape website data

Important Notes

  • URL must be valid and accessible
  • Response includes structured data from the target website
  • Images and links are extracted with their attributes
  • Text content is extracted from paragraph elements
  • Some websites may block scraping attempts
  • Response time may vary based on website size and complexity
