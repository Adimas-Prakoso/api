Available Endpoints

Search by Type

GET /doujindesu/searchtype

Query Parameters:

page Page number (default: 1)
type Content type (default: Doujinshi). Available: Doujinshi, Manga, Manhwa

Example Requests:

GET /doujindesu/searchtype GET /doujindesu/searchtype?page=1&type=Manhwa

Example Response:

{
  "data": [
    {
      "thumbnail": "https://doujindesu.tv/uploads/xxxx.jpg",
      "title": "Example Title",
      "type": "Manhwa",
      "chapter": "Chapter 1",
      "updateTime": "NEW"
    },
    // ... more items
  ]
}

Response Codes

200 Success - Returns array of manga/doujin data
400 Bad Request - Invalid parameters
500 Server Error - Something went wrong on the server
