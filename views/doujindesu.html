Search Doujin

GET /doujindesu/search ‚ñº

Query Parameters:

page Nomor halaman (default: 1)
title Judul doujin
author Nama author
character Nama karakter
status Status doujin (contoh: Publishing)
type Tipe doujin (contoh: Manga)
genre Genre dalam format array [genre1, genre2] atau comma-separated genre1,genre2

Example Requests:

GET /doujindesu/search GET /doujindesu/search?title=example&type=Manga GET /doujindesu/search?title=example&type=Manga&genre=[Ahegao,Romance]

Example Response:

{
  "data": [
    {
      "thumbnail": "https://doujindesu.tv/uploads/xxxx.jpg",
      "title": "Example Title",
      "type": "Manga",
      "rating": "4.5",
      "status": "Publishing"
    },
    // ... more items
  ]
}

Response Codes

200 Success - Returns array of manga/doujin data
400 Bad Request - Invalid parameters
500 Server Error - Something went wrong on the server
